Cento Mobile iPhone Install

Use this page as the short checklist for opening the PWA on iPhone and adding it to the Home Screen.

Path PWA

Current LAN Link

http://10.0.0.56:47918/ workspace/runs/agent-work/18/state/token.txt

The gateway must be running on Linux with --host 0.0.0.0. The iPhone must be on a network path that can reach 10.0.0.56.

Start Gateway

  1. From the Cento repo, run python3 workspace/runs/agent-work/18/server.py --host 0.0.0.0 --port 47918.
  2. Keep the terminal session open while using the iPhone PWA.
  3. Copy the printed gateway token, or read it from workspace/runs/agent-work/18/state/token.txt.

Install On iPhone

  1. Open Safari on the iPhone.
  2. Go to http://10.0.0.56:47918/.
  3. Paste the gateway token into the setup panel and tap Save.
  4. Confirm Dashboard shows queue, jobs, agents, and activity.
  5. Tap Share, then Add to Home Screen.
  6. Launch Cento from the Home Screen icon.

Smoke Test

  • Dashboard loads without token prompt after first setup.
  • Tap an issue card from Dashboard queue or Issues list.
  • Issue detail opens near the top.
  • Activity tab says Redmine journal endpoint is pending.
  • Actions screen exposes only allowlisted live actions.
