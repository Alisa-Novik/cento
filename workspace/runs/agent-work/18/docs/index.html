Agent-work #18

iPhone Cento App

First phone-visible PWA/mobile gateway delivery for the dark orange Cento operations cockpit.

Review Ready

Original Request

The requested target was a native-feeling iPhone control surface for Cento: dashboard, Redmine issue views, job execution, live logs, artifacts, agents, quick actions, activity, and settings. The supplied reference was a dark industrial mobile UI with orange primary actions and compact operational cards.

Dashboardhealth, queue, activity
Issuesdetails, activity, relations
Jobssteps, logs, artifacts
Agentsnode state and roles
Actionsallowlisted commands

Delivered Scope

  • Responsive PWA shell with dark/orange Cento styling.
  • Local gateway at workspace/runs/agent-work/18/server.py.
  • Token setup for browser and iPhone use.
  • Dashboard, Redmine queue, jobs, agents, and recent activity.
  • Allowlisted quick actions only: cluster.status and notify.test.
