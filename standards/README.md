# Standards

These documents define repo-wide conventions for `cento`.

Current standards:

- [tui.md](./tui.md)
  Conventions for interactive terminal applications.
- [tool-registration.md](./tool-registration.md)
  Conventions for registering and documenting tools.
- [mcp.md](./mcp.md)
  Conventions for repo-root MCP configuration and tool-call guidance.

Use these before adding new user-facing commands or terminal surfaces.
